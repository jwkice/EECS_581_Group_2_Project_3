### For Development

**Install Node.js (if not already installed):**  
Download and install from:
```bash
https://nodejs.org/en/download
```



**Install FastAPI**

```bash
pip install "fastapi[standard]"
```

**Navigate into `frontend/src`:**

```bash
cd frontend/src
```

**To install dependencies:**

```bash
npm install
```

**To run frontend:**

```bash
npm run dev
```

**To run backend:**

```bash
python main.py
```
